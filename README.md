# Production Control API

Веб-приложение для управления сменными заданиями на производстве с поддержкой асинхронной обработки задач, аналитики и webhook-интеграций.

---

## Технологический стек

- Python 3.11+
- FastAPI
- SQLAlchemy (async)
- PostgreSQL
- Redis
- RabbitMQ
- Celery + Celery Beat
- MinIO (S3)
- Docker + Docker Compose

---

## Архитектура

Проект построен по слоистой архитектуре:

- API слой (FastAPI routers)
- Service слой (бизнес-логика)
- Repository слой (доступ к данным)
- Unit of Work (управление транзакциями)
- Celery workers (асинхронные задачи)

---

## Функциональность

### Управление партиями (Batches)
- создание сменных заданий
- получение списка партий
- получение детальной информации
- статистика по партии
- закрытие партии

---

### Продукция (Products)
- агрегация продукции (sync / async)
- обработка уникальных кодов

---
