# Production Control API

Веб-приложение для управления сменными заданиями на производстве с поддержкой асинхронных задач, файлового хранилища и webhook-интеграций.

## Технологический стек

- Python 3.11+
- FastAPI
- SQLAlchemy (async)
- PostgreSQL
- Redis
- RabbitMQ
- Celery + Celery Beat
- MinIO (S3)
- Docker + Docker Compose

---

## Функциональность

### Управление партиями (Batches)
- Создание сменных заданий
- Получение списка партий
- Статистика по партии
- Закрытие партии

---

### Продукция (Products)
- Агрегация продукции (sync / async)
- Обработка уникальных кодов

---

### Асинхронные задачи
- Celery worker + Redis backend
- Отслеживание статуса задач `/tasks/{task_id}`

---

### Работа с файлами
- Генерация отчетов (Excel)
- Экспорт партий
- Импорт партий из CSV
- Хранение файлов в MinIO

---

### Аналитика
- Статистика по производству
- Производительность команды
- Расчет темпов производства

---

### Webhooks
- Подписка на события
- Поддержка событий:
  - batch_created
  - batch_closed
- Логирование доставок
- Повторные попытки отправки через Celery
